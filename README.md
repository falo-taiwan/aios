# Public Draft References

這個目錄存放的是 FALO 的公開草稿頁與 reference page。

這些檔案的定位是：

- draft base
- reference page
- narrative baseline
- GitHub public draft asset

這些檔案 **不是**：

- CM truth
- source record
- canonical article
- 正式 article record

原則：

- 可作為 GitHub Pages 的公開草稿頁
- 可作為 FW template / block system / landing page 的參考資產
- 若未來升格為正式內容，應另建正式 article / page record
- 不應直接把這裡的 reference 檔案視為 truth
